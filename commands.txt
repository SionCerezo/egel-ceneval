Llenar (seeding) base de datos con los catalogos
    -> php artisan db:seed --class=CatalogsSeeder

Instalacion de laravel UI
    -> composer require laravel/ui
Generacion de autenticacion de laravel UI
    -> php artisan ui bootstrap --auth
        * <bootstrap> indica que se trabajar con el framwork de Bootstrap


Crear un Provider
    -> php artisan make:provider HelpersServiceProvider
